# react-timer-advance

A timer application with keypad and separate input for hours, minutes and seconds using [ReactJS](https://reactjs.org/).

Check [here](https://sample-react-projects.web.app/react-timer-advance) for live demo.

## Developed With

- [Node.js](https://nodejs.org/en/) - Javascript runtime
- [ReactJS](https://reactjs.org/) - A javascript library developed by Facebook
- [Create React App](https://facebook.github.io/create-react-app/) - Ready to use ReactJS boilerplate
- [SASS](https://sass-lang.com/) - A CSS preprocessor

## Getting Started

### Installing

Run the command below to install node packages

```
npm install
```

or

```
yarn install
```

### Running

To start the application run the command below, open a browser and go to `http://localhost:3000`

```
npm start
```

or

```
yarn start
```
